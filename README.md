# DP_FDR

This repository contains the code and supporting files for the paper:

**Xintao Xia and Zhanrui Cai (2023).  
"Adaptive False Discovery Rate Control with Privacy Guarantee."  
Journal of Machine Learning Research, 24(252):1–35.**

The repository includes the original code used in the paper, data and figures from the numerical experiments, and correction materials associated with the erratum. The correction folder also contains the corrected code and updated results after correction, which are almost identical to the original results.
